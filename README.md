# Customer Behavior Analysis

##  Project Overview

This project analyzes customer shopping behavior to identify key revenue drivers, customer segments, and opportunities for business optimization.

The analysis is performed using Python, PostgreSQL, and Power BI.

---

##  Tools & Technologies

* Python (Pandas, Data Cleaning)
* PostgreSQL (SQL Analysis)
* Power BI (Dashboard Visualization)

---

##  Project Workflow

1. Data Cleaning & Preparation (Python)
2. Data Storage (PostgreSQL)
3. Data Analysis (SQL Queries)
4. Data Visualization (Power BI Dashboard)

---

##  Key Features

* Customer segmentation (New, Returning, Loyal)
* Revenue analysis by gender, age group, and category
* Discount and subscription impact analysis
* Product performance insights

---

##  Key Insights

* Loyal customers contribute the majority of revenue
* Subscription model does not significantly increase spending
* Discounts are widely used but may not be optimized
* Young adults are the highest revenue-generating segment

---

##  Business Recommendations

* Improve customer retention strategies
* Redesign subscription offerings
* Optimize discount usage
* Target high-value customer segments
